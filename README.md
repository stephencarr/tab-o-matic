## 🚀 Installation

1. 📦 Download and extract the zip file
2. 🌐 Open Chrome and go to `chrome://extensions/`
3. 🔧 Enable "Developer mode" (toggle top-right)
4. ➕ Click "Load unpacked"
5. 📁 Select the `tab-o-matic` folder
6. ✅ Done! Open a new tab to see your dashboard

## 📖 Usage

### 📌 Stashing a Tab

**Method 1: ⌨️ Keyboard Shortcut**
- Press `Ctrl+Shift+S` (or `Cmd+Shift+S` on Mac)
- Add notes and/or reminder
- Click "Stash & Close"

**Method 2: 🖱️ Right-Click**
- Right-click anywhere on the page
- Select "Stash this tab..."
- Add notes and/or reminder
- Click "Stash & Close"

**Method 3: 🔌 Extension Icon**
- Click the Tab-o-matic icon in toolbar
- Add notes and/or reminder
- Click "Stash & Close"

### 🎯 Using the Dashboard

**📊 Recent Activity (Default View):**
- 🆕 Open a new tab to see your dashboard
- 👀 Browse recent history with date grouping
- 🔍 See your most visited sites with actual visit counts
- 🖱️ Click any item to open it
- 🌐 Use domain filters to narrow results
- ⏰ Stashed tabs show reminder indicator

**📚 Stash Management:**
- ➡️ Click "View Stashed Tabs" in sidebar
- 📋 See all stashed tabs organized by urgency
- ✅ Use category filters (Overdue, Today, This Week, No Reminder)
- 🔓 Click "Open" to open and remove from stash
- ✏️ Click "Edit" to change notes or reminder
- 🗑️ Click "Delete" to remove without opening
- ☑️ Use checkboxes for bulk actions

### 🗂️ Organizing Tabs

**🔗 Group by Domain:**
- Click "Group by Domain" button in sidebar
- All tabs in current window will be organized into color-coded tab groups
- Each domain/subdomain gets its own group
- Perfect for organizing research or work sessions

**🪟 Split to Windows:**
- Click "Split to Windows" button in sidebar
- Creates a separate window for each domain
- Great for focusing on specific projects
- Keeps related tabs together

**📑 Ungroup All:**
- Click "Ungroup All" button in sidebar
- Merges all tabs from all windows into one window
- Removes all tab groups
- Returns to a clean, unified tab layout

### ✏️ Editing Stashed Tabs

1. 📂 Navigate to Stash Management view
2. ✏️ Click "Edit" on any stashed tab
3. 📝 Modify notes
4. ⏰ Change reminder (or clear it)
5. 💾 Click "Save"

## ⌨️ Keyboard Shortcuts

- `Ctrl+Shift+S` (Mac: `Cmd+Shift+S`) - 📌 Stash current tab

💡 Want more shortcuts? Go to `chrome://extensions/shortcuts` to customize.
